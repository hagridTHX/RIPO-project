# RIPO-project

## Jak uruchomić:

### Wymagania:
- Python 3.8 lub wyżej
- Kamera internetowa

### Instalacja (Windows):

1. Utwórz virtual environment:
```
python -m venv venv
```

2. Aktywuj environment:
```
.\venv\Scripts\activate
```

3. Zainstaluj zależności:
```
pip install -r requirements.txt
```

4. Uruchom aplikację:
```
python main.py
```

5. Wyjście:
```
Naciśnij 'q' aby zamknąć
```
